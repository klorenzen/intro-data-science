Let's revisit our questions about the burrito data set. Let's answer this using Excel. We can do most of our data verbs if we turn our data set into a table. For group and aggregate we will need to use a pivot table. We can pull our variables into rows, columns, value - where we can choose our aggregate function. This is not the full extent of what a pivot table can do but is helpful for us for quick data analysis.
